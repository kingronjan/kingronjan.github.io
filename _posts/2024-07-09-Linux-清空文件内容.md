- 使用 `vi/vim` 命令打开文件后，输入 `"%d"` 清空，后保存即可。但当文件内容较大时，处理较慢：
   ```bash
   vim file_name
   :%d
   :wq
   ```



- 使用 `cat` 命令情况：
   ```bash
   cat /dev/null > file_name
   ```



- 使用 `echo` 命令清空，此时会在文件中写入一个空行 `\n`：
   ```bash
   echo "" > file_name
   ```
